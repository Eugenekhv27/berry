Как это работает?

Перед совершением покупки, если покупатель согласен участвовать в программе BONUS CLUB, продавец уточняет номер телефона покупателя и вводит ее в систему. С помощью SMS подтверждается, что данный телефон принадлежит этому покупателю. Индефикация участников в системе происходит по номеру телефона.

При совешении покупателем покупки, ему начисляется 5% от бонусной суммы, 3% начисляется участнику, который привлек текущего покупателя, и 2% тому кто привлек последнего.

Покупатель может зайти на страничку бонус клуба, и узнать сколько у него начисленно баллов по каждому магазину.

API для интеграции с кассой

Начислить бонусы покупателю

putPoints/:userTel/:uuid/:rubSum/:pointsSum/:fromTel Method="GET"

:userTel - телефон покупателя
:uuid - уникальный индефикатор документа при совершении продажи (обычно чек)
:rubSum - сумма покупки
:pointsSum - сумма бонусных баллов, определенная для данного товара. От этой суммы идет распределение на 5% ,3% и 2%.
:fromTel - номер телефона,человека который привлек покупателя (в случае, если покупатель совершает первую покупку)

Получить баланс

getBalance/:userTel/:fromTel Method="GET"

:userTel - телефон покупателя
:fromTel - номер телефона,человека который привлек покупателя (в случае, если покупатель совершает первую покупку)

Оплата товара за счет бонусов

payByPoints/:userTel/:uuid/:points Method="GET"

:userTel - телефон покупателя
:uuid - уникальный индефикатор документа при совершении продажи (обычно чек)
:points - количестово бонусов к списанию

Запросить смс с проверочным кодом для индефикации номера телефона

genCode/:userTel Method="GET"

:userTel - телефон покупателя

Проверить код, который был получен по SMS

checkCode/:userTel/:code Method="GET"

:userTel - телефон покупателя
:code - проверочный код из SMS
